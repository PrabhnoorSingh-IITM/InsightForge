How It Works

1

Data Ingestion (`datasets/raw`)

Multiple CSV/JSON files (catalog, reviews, pricing, signals) are parsed and normalized into a standard taxonomy to ensure structural integrity before analysis.

2

Prompt Engineering (`research_agent.py`)

The Python engine combines the user's business goal (e.g., Growth), the target SKU, and contextual domain memory into a highly structured, instruction-heavy system prompt.

3

LLM Reasoning Integration

The FastAPI backend calls the Large Language Model. The model doesn't just restate data; it evaluates price elasticity, customer sentiment gaps, and competitive threats based on the prompt.

4

Decision Output

A structured JSON payload returns featuring Confidence Scores, Risk Arrays, and a final Markdown report containing prioritized Next Actions for the user.
